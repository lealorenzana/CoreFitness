# G-FITNESS CORE - Gym Management System

A comprehensive gym management system with member and admin applications.

## 🚀 Applications

### 📱 G-FITNESS MEMBER APP
**Location:** `g-fitness-member/`  
**Purpose:** Member portal for gym members  
**Features:**
- Login & Registration
- View membership details
- Book classes (HIIT, Yoga, Strength Training, etc.)
- View booking history
- Track attendance
- View payment history
- Renew membership
- Chatbot support
- Event registration

**To Run:**
```bash
cd g-fitness-member
npm install
npm run dev
```
**URL:** http://localhost:5173/

---

### 🏋️ G-FITNESS ADMIN
**Location:** `g-fitness-admin/`  
**Purpose:** Admin portal for gym management  
**Features:**
- Dashboard with analytics
- Member management (Add, Edit, Delete)
- Attendance tracking (QR + Manual)
- Payment tracking
- Booking history
- Revenue analytics
- Retention monitoring
- Trainer management
- Schedule management
- Export to CSV

**To Run:**
```bash
cd g-fitness-admin
npm install
npm run dev
```
**URL:** http://localhost:5175/

## 🎯 Quick Start

1. **Install dependencies for both apps:**
```bash
cd g-fitness-member && npm install
cd ../g-fitness-admin && npm install
```

2. **Run both applications:**
```bash
# Terminal 1 - Member App
cd g-fitness-member
npm run dev

# Terminal 2 - Admin App
cd g-fitness-admin
npm run dev
```

3. **Access the applications:**
- Member App: http://localhost:5173/
- Admin App: http://localhost:5175/
