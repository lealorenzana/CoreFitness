## 🎯 KEY TALKING POINTS

### Problem Statement:
- ❌ Manual record-keeping → inefficient, error-prone
- ❌ Paper attendance logs → difficult to track
- ❌ Payment monitoring → lacks transparency
- ❌ Member communication → fragmented
- ❌ No data insights → reactive management

### Solution Highlights:
- ✅ Centralized Management → all operations in one platform
- ✅ QR Attendance → automated, accurate, secure
- ✅ NLP Chatbot → reduces repetitive inquiries
- ✅ Rule-Based Analytics → proactive retention
- ✅ RBAC → secure role-based access
- ✅ Cross-Platform → unified web & mobile experience

### Technical Stack:
- Frontend: React.js + TypeScript + Vite
- Styling: Tailwind CSS + Framer Motion
- Data: SharedStorage (localStorage for prototype)
- Future: Node.js + Express + MySQL + Firebase
- Quality: ISO 25010 compliance

## 🎤 QUICK SCRIPTS

### Opening (30 seconds):
> "Good [morning/afternoon]. Today I will demonstrate **CORE Fitness**, a comprehensive gym management system designed to digitalize and streamline fitness center operations in Mamburao, Occidental Mindoro. Traditional gyms rely on manual record-keeping, paper logs, and spreadsheets—leading to inefficiency, errors, and lost information. CORE Fitness solves these problems through a cross-platform digital ecosystem. Let's begin with the Admin Dashboard."

### Transition Admin → Member:
> "Now let's switch to the Member Mobile Application. This is what gym members use on their smartphones to access services, track progress, and communicate with the gym."

### Transition Member → Trainer:
> "Trainers also have their own dedicated interface. Let me show you how trainers manage their assigned members and sessions."

### Closing (30 seconds):
> "To summarize, CORE Fitness addresses traditional gym management challenges through QR-Based Attendance, NLP Chatbot, Rule-Based Analytics, Cross-Platform Integration, and Role-Based Access Control. The system was evaluated using ISO 25010 standards and validated through user acceptance testing with local gym stakeholders. Thank you for your attention. I'm now open to questions."
